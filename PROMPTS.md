# CONTEXT
Перед тобой - требования для нового веб-приложения:

«Мини-анкета», - мини-приложение, состоящее из backend-части и небольшого frontend-интерфейса.

Требуемый функционал: 

Backend предоставляет два метода API: 

1. GET /questions — возвращает список вопросов анкеты (жёстко заданных, 3–5 шт.). 
2. POST /answers — принимает ответы пользователя и сохраняет их в памяти (например, в slice/array/map). 

Frontend:

1. загружает вопросы с backend; 
2. отображает их в HTML/JS интерфейсе; 
3. отправляет заполненные ответы через POST /answers; 
4. показывает пользователю сообщение «Спасибо!» после отправки. 

# RULES AND RESTRICTIONS
1. Для Frontend используй только HTML, CSS и JavaScript
2. Для Backend используй Go
3. Не используй БД, все сущности просто храни их в памяти на Backend.
